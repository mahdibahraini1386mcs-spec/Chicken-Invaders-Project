## ⚙️ راهنمای اجرا و وابستگی‌ها (Setup & Execution)
- **زبان و فریم‌ورک:** `Java` | `Swing 2D GUI`
- **وابستگی اصلی (Dependency):** درایور `sqlite-jdbc.jar`
- **دستورالعمل راه‌اندازی:**
    1. سورس‌کدها را در محیط توسعه محبوب خود (مثل **IntelliJ IDEA**) باز کنید.
    2. فایل درایور SQLite (`sqlite-jdbc.jar`) را به بخش `Libraries` یا `Dependencies` پروژه متصل (Add) کنید.
    3. کلاس `Main` بازی را Build و Run کنید. سیستم به صورت خودکار دیتابیس را برای شما می‌سازد!

## 🗄️ معماری پایگاه داده (Database Structure)
قلب تپنده بازی روی یک دیتابیس **SQLite** (سبک، لوکال و بدون نیاز به نصب سرور) طراحی شده است.

- **مسیر دیتابیس:** فایل `game_data.db` به محض اولین اجرای موفق، به صورت خودکار در مسیر روت (Root) پروژه تولید می‌شود.
- **ساختار جداول (Tables):**
    - 🛡️ **جدول `users`:** مدیریت حساب‌های کاربری (نام کاربری، رمز عبور هش‌شده) و ذخیره سفینه فعال خلبان در ستون `active_plane`.
    - 🏆 **جدول `scores`:** ثبت تاریخچه بازی‌ها (نام کاربر، امتیاز، مرحله نهایی). از این جدول برای استخراج زنده بالاترین امتیاز (High Score) جهت نمایش در منو و خرید سفینه‌ها استفاده می‌شود.
    - ⚙️ **جدول `settings`:** ذخیره ترجیحات شخصی هر کاربر برای تنظیمات بازی (وضعیت فعال/غیرفعال بودن موسیقی و افکت‌های صوتی).
